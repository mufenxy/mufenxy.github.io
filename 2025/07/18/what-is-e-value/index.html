What is e-value?

发表于 2025-07-18

E-value: 观察性研究中的敏感性分析新方法

1. 研究背景与问题

1.1 观察性研究的局限性

在医学和社会科学研究中，我们经常需要探索某种干预措施（如药物治疗、生活方式改变）对健康结果的影响。虽然随机对照试验（RCT）被认为是评估因果关系的"金标准"，但在许多情况下，进行RCT是不可行或不道德的。因此，观察性研究成为了重要的研究方法。

然而，观察性研究面临一个核心挑战：混杂因素（confounding）。

什么是混杂因素？
混杂因素是指同时影响干预措施和结果的第三个变量。它可能导致我们错误地认为存在因果关系，而实际上观察到的关联可能完全由混杂因素解释。

举例说明：
假设我们研究"喝咖啡是否导致心脏病"。我们发现喝咖啡的人心脏病发病率更高，但这可能是因为：

  • 喝咖啡的人更可能吸烟
  • 喝咖啡的人工作压力更大
  • 喝咖啡的人睡眠质量更差

在这种情况下，吸烟、工作压力、睡眠质量就是潜在的混杂因素。

1.2 未测量混杂的挑战

研究者通常会尽力控制已知的混杂因素，但总是存在未测量混杂的可能性——即那些我们没有观察到或没有考虑到的混杂因素。这就产生了一个根本性问题：

“即使我们控制了所有已知的混杂因素，如何评估未测量的混杂因素是否足够强大，以至于能够完全解释观察到的关联？”

2. E-value的定义与核心概念

2.1 E-value的定义

**E-value（证据值）**是VanderWeele和Ding于2017年提出的一个新指标，其定义为：

在风险比尺度上，一个未测量的混杂因素需要与干预措施和结果都产生的最小关联强度，才能完全解释掉观察到的特定干预-结果关联（在已控制测量协变量的条件下）。

2.2 用通俗语言理解E-value

想象E-value是一个"抗干扰能力指标"：

  • E-value越大：表示需要非常强的未测量混杂因素才能解释掉观察到的效应，研究结果更可靠
  • E-value越小：表示相对较弱的未测量混杂因素就能解释掉观察到的效应，研究结果较不可靠

3.2 母乳喂养与儿童白血病案例

研究发现：
RR = 0.80（95%置信区间：0.71-0.91）

计算步骤：

  1. 因为RR < 1，先计算RR* = 1/0.80 = 1.25
  2. E-value = 1.25 + √[1.25 × (1.25 - 1)] = 1.8

结果对比：

  • 估计值的E-value = 1.8（相对较小）
  • 置信区间上限的E-value = 1.4（非常小）

解释：
相比呼吸道感染的案例，这里的证据要弱得多。一个与母乳喂养和儿童白血病都有1.4倍关联的未测量混杂因素就足以将置信区间推向包含无效应的区域。

5. E-value的优势与特点

5.1 相比传统敏感性分析的优势

传统方法的局限性：

  1. 主观性强：研究者需要主观设定混杂因素的强度参数
  2. 假设较多：通常假设混杂因素是二元的，或只有一个未测量混杂因素
  3. 复杂性高：需要复杂的计算和多个参数设定

E-value的优势：

  1. 客观性：不需要研究者主观设定参数
  2. 假设最少：不对未测量混杂因素的结构做任何假设
  3. 计算简单：只需要一个公式即可计算
  4. 解释直观：提供直接的数值来评估证据强度

5.2 E-value与P值的区别

特征 P值 E-value
评估内容 是否存在关联 关联的因果证据强度
样本量依赖 高度依赖 主要依赖效应大小
混杂因素 不考虑 直接评估
解释 统计显著性 因果推断的稳健性

重要观点：

  • 大型研究可能有很小的P值，但如果效应量小，E-value也会很小
  • 小样本研究可能有较大的P值，但如果效应量大，E-value可能很大
  • 两者提供不同维度的证据，都应该报告

6. E-value的解释与应用指南

6.1 解释原则

6.1.1 上下文依赖性

E-value的解释必须考虑研究的具体背景：

  • 控制变量的质量：如果研究控制了多个高质量的协变量，相同的E-value意味着更强的证据
  • 领域知识：在生物医学和社会科学研究中，2-3倍的效应量偶尔出现但不常见

6.1.2 效应量的现实性评估

研究者应该问：

  • 在我的研究领域中，是否存在能够产生如此强关联的未测量因素？
  • 这样的因素是否合理？

6.2 报告标准

建议的报告格式：
“观察到的风险比X.X可能被一个未测量混杂因素完全解释掉，但该混杂因素需要在已控制测量协变量的基础上，同时与治疗和结果都产生Y.Y倍的关联强度。比这更弱的混杂因素无法做到这一点。置信区间可能被同时与治疗和结果都产生Z.Z倍关联的未测量混杂因素推向包含无效应，但更弱的混杂因素无法做到这一点。”

6.3 应用建议

6.3.1 研究设计阶段

  • 在计划观察性研究时，可以为假设的效应量计算E-value
  • 如果预期的E-value很小，可能需要等待资源充足时进行随机对照试验

6.3.2 结果报告阶段

  • 所有评估因果关系的观察性研究都应报告E-value
  • 应该报告估计值和置信区间的E-value
  • 可以结合系统综述报告E-value

7. 局限性与注意事项

7.1 E-value的局限性

7.1.1 不能解决所有偏倚问题

E-value只关注未测量混杂，不能解决：

  • 测量误差
  • 选择偏倚
  • 缺失数据
  • 结果的选择性报告

7.1.2 不保证实际解释

E-value不能保证如果存在具有特定强度的混杂因素，就一定能解释掉效应。它只是表明这种可能性存在。

7.1.3 稀有混杂因素的影响

罕见的未测量混杂因素不会像常见的混杂因素那样产生大的偏倚。

7.2 常见误解

7.2.1 E-value不是阈值

  • E-value是连续测量，不应该设定任意的截断点
  • 不应该仅凭E-value的大小决定是否发表研究

7.2.2 小E-value不意味着无效应

  • 小E-value只表明因果证据较弱
  • 证据不足不等于证据表明无效应

8. 实际应用与未来发展

8.1 当前应用状况

自2017年发表以来，E-value已经在多个领域得到应用：

  • 流行病学研究
  • 公共卫生政策评估
  • 临床医学研究
  • 社会科学研究

8.2 软件支持

目前已有多个统计软件包支持E-value计算：

  • R语言的EValue包
  • Stata的相关命令
  • 在线计算器

8.3 未来发展方向

8.3.1 方法学扩展

  • 扩展到更复杂的研究设计
  • 结合其他敏感性分析方法
  • 发展针对特定偏倚类型的变体

8.3.2 应用推广

  • 期刊编辑政策的改变
  • 研究报告标准的更新
  • 教育培训的加强

9. 结论与启示

9.1 E-value的重要意义

E-value代表了敏感性分析领域的重要进展：

  1. 简化了复杂概念：将敏感性分析从复杂的参数设定简化为单一数值
  2. 提高了研究质量：促使研究者更仔细地考虑因果推断的稳健性
  3. 改善了科学交流：为评估观察性研究证据提供了标准化工具

9.2 对研究实践的影响

9.2.1 对研究者的建议

  • 在所有旨在评估因果关系的观察性研究中报告E-value
  • 不要仅依赖P值评估因果证据
  • 将E-value作为研究设计的考虑因素

9.2.2 对期刊编辑的建议

  • 强烈鼓励或要求报告E-value
  • 培训审稿人理解和评估E-value
  • 在发表决策中综合考虑统计显著性和因果证据强度

9.3 科学发展的长远影响

E-value的广泛应用可能会：

  • 减少观察性研究结果的过度解释
  • 提高科学界对因果推断的整体水平
  • 促进更严谨的研究设计和报告标准

正如作者所说：

“科学的目标不是发表，而是集体努力尽可能地接近真理。我们希望E-value能够在这方面发挥作用。”

通过引入E-value，我们向建立更加稳健和可靠的观察性研究体系迈出了重要一步。这不仅有助于提高单项研究的质量，更重要的是，它为整个科学界提供了一个评估因果证据的标准化工具，最终将有助于科学知识的积累和进步。
