基于神经网络拓展的泊松回归模型

发表于 2025-07-11 更新于 2025-07-18

NPR using NN

tag:

  • GLM
  • Poisson regression
  • Nonlinear regression

Summary

发现了基于多层感知器（一种神经网络）的泊松回归模型新扩展

1.使用了一种新的方法：六种不同方案的神经网络回归模型，在模拟数据集（一个线性案例和两个非线性案例）上比较性能。
2.在线性假设成立时，PR表现良好，非线性下，using NN 显著提升预测效果。

Intro

神经网络以其高度的灵活性，在预测能力方面表现优异，甚至可以与其他先进模型(GAM\CART\MARS)相媲美，神经网络在统计建模中的融合与创新，展示了其在不同类型回归模型和生存分析中的巨大潜力。GLM应用广泛。

本文中，将线性泊松回归扩展至神经网络泊松回归，并通过模拟数据对比其与线性泊松回归的性能表现。

Methods

Neural networks

A MLP consists of one input layer of units,one output layer of units and one or more layers of ‘hidden’ units.

隐藏层：对输入进行加权求和，加上偏置项，再通过激活函数处理

Model fitting and gradient calculation

模拟比较了不同模型的性能，惩罚似然误差准则函数

对隐藏层单元数固定的模型进行拟合，以防止过拟合。

to identify the number of units :
$$
\begin{aligned}

&\mathrm{AIC}=-2\times\text{Loglikelihood}+2\times m\&\mathrm{BIC}=-2\times\text{Loglikelihood}+m\times\log(N)

\end{aligned}
$$
MSE:

不同模型（包括2、3、4、5、10、20个隐藏单元）的预测结果按均方误差（MSE）进行排序。为了节省计算时间，模拟中将权重衰减参数预设为0.012，该值是通过对不同权重衰减参数的实证研究后选择的。

线性泊松回归（“·”）：在较低lambda值时表现尚可，但随着lambda值增加，预测偏差逐渐增大，特别是在高值区域出现明显的系统性偏差

Conclusion

本文提出了一种基于神经网络的泊松回归非线性扩展方法，并通过模拟数据比较了两类模型的性能。

传统统计方法泊松回归能够刻画自变量与因变量间的线性关系，而人工神经网络结构则能增强对自变量非线性关系的建模灵活性。

神经网络在处理非线性关系时比线性Poisson回归更具预测准确性，但需要更复杂的参数设置和编程。
